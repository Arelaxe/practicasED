Descripción detallada

template<class T>
class Cola_max< T >

Una instancia c de la clase Cola_max (implementada con vector dinámico) es un objeto que contiene un elemento de la clase vector de STL, que funciona como envoltorio para datos de tipo Pareja.

Una instancia c de la clase Cola_max (implementada con pilas) es un objeto que contiene dos elementos de la clase stack de la STL. La primera pila se usará para la entrada de datos y la segunda para la salida de datos.
