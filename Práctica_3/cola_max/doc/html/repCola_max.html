COLA_MAX
Rep de Cola_max

Invariante de la representación

  • El vector que envuelve los datos debe ser en todo momento una cola correcta y seguir el orden FIFO en la inserción y extracción de datos

Función de abstracción

Un objeto válido rep del TDA Cola_max (con vector dinámico) debe seguir la siguiente representación:

  • Debe tener un vector que almacena los datos, sobre el que se harán todas las operacines de la cola

Invariante de la representación

  • La pila entrada sólo se utiliza para operaciones de entrada de datos o como pila auxiliar
  • La pila salida sólo se utiliza para operaciones de salida o como pila auxiliar
  • Las inserciones y extracciones deben seguir el orden FIFO

Función de abstracción

Un objeto válido rep del TDA Cola_max (con pilas) debe seguir la siguiente representación:

  • Debe tener una pila entrada para la entrada de datos
  • Debe tener una pila salida para la salida de datos
